Правила для преподавателей

Назад к панели
Общие правила работы
Основные принципы:
  1. Профессионализм: Всегда поддерживайте высокий уровень профессиональной этики
  2. Конфиденциальность: Не разглашайте личную информацию студентов
  3. Объективность: Оценивайте обращения беспристрастно и справедливо
  4. Ответственность: Берете на себя ответственность за качество ответов
  5. Коллегиальность: При необходимости консультируйтесь с коллегами
Временные рамки ответов
Срочные обращения (24 часа):
  • Академические вопросы
  • Проблемы с доступом к материалам
  • Технические вопросы
  • Жалобы на качество обучения
Обычные обращения (48 часов):
  • Общие вопросы
  • Консультации
  • Предложения по улучшению
  • Информационные запросы
Важно: Если вы не можете ответить в установленные сроки, обязательно уведомите студента о причине задержки.
Правила коммуникации
Требования к ответам:
✅ Обязательные элементы:
  • Приветствие по имени
  • Благодарность за обращение
  • Четкий ответ на вопрос
  • Конкретные рекомендации
  • Предложение дополнительной помощи
  • Профессиональная подпись
❌ Запрещено:
  • Использование неформального языка
  • Эмоциональные реакции
  • Обвинения в адрес студента
  • Разглашение личной информации
  • Игнорирование вопросов
  • Перенос ответственности
Правила управления статусами
Когда и как менять статусы:
Статус Когда использовать Действия
Ожидает Новое обращение Прочитать и проанализировать
В работе Начали работу с обращением Подготовить ответ
Завершено Ответ отправлен Проверить удовлетворенность
Отклонено Обращение не по компетенции Объяснить причину
Правила конфиденциальности
Защита персональных данных:
Строго запрещено:
  • Разглашать личную информацию студентов третьим лицам
  • Использовать данные студентов в личных целях
  • Передавать доступ к системе другим лицам
  • Сохранять обращения вне системы
  • Обсуждать обращения в публичных местах
Обязательно: Все обращения должны обрабатываться только через официальную систему.
Правила эскалации
Когда передавать обращение вышестоящим:
Срочная эскалация:
  • Жалобы на дискриминацию
  • Проблемы с безопасностью
  • Нарушения академической честности
  • Конфликты между студентами
Обычная эскалация:
  • Вопросы не по вашей компетенции
  • Сложные академические споры
  • Системные проблемы
  • Предложения по изменениям
Процедура эскалации: Уведомите студента о передаче обращения и укажите сроки ответа.
Контроль качества
Критерии оценки качества ответов:
Качественный ответ включает:
  • Полноту информации
  • Точность данных
  • Понятность изложения
  • Конструктивность
  • Профессионализм
  • Соблюдение сроков
Регулярная самопроверка:
  • Анализ статистики ответов
  • Оценка удовлетворенности
  • Изучение повторяющихся проблем
  • Обновление знаний
  • Консультации с коллегами
